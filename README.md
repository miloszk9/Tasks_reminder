# Tasks_reminder

Web app made to keep and share task list.

## Used frameworks / tools
* Django
* PostreSQL
* psycopg2 
* Bootstrap

## Installation
```sh
$ pip install django psycopg2
```

## Migrate:
Migrate tables from database.sql file to postgres database
```sh
$ python manage.py migrate
```

## Usage

```sh
$ python manage.py runserver
```
